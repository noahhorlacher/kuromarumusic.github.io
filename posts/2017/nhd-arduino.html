How to connect a NHD C12864A1ZFSBFBWH to an Arduino

Noah Horlacher, 5 October 2017

I bought a Newhaven Display (C12864A1ZFSBFBWH) for my Arduino and had no idea how to connect and program it.
The datasheet did only offer technical information, but no tutorial whatsoever.
So here it is: How to connect a NHD C12864A1ZFSBFBWH to an Arduino.

In this tutorial, you'll learn how to connect a Newhaven Display C12864A1ZFSBFBWH to an Arduino and display a simple "Hello World!".


You'll need:

  • Arduino (I'm using the Arduino Mega 2560)
  • Newhaven Display C12864A1ZFSBFBWH (Buy)


Connect your display.

  1. Digital Pin 10 to LCD Pin 1
  2. Digital Pin 9 to LCD Pin 2
  3. Digital Pin 8 to LCD Pin 4
  4. Digital Pin 7 to LCD Pin 5
  5. Digital Pin 6 to LCD Pin 6
  6. 3.3V output to LCD Pin 3 and 12
  7. LCD Pin 7 to ground


Install the library.

Inside the Arduino IDE, go to Sketch > Include Library > Manage Libraries...
A window will pop up. Click the search bar and search for "U8g2".
Select the newest version and click on "Install".
Alternatively, the library can be downloaded here and added at "Sketch > Include Library > Add .ZIP Library" in the Arduino IDE.


Run the code.

The following code will display a simple "Hello World!" on your Newhaven Display. Check the comments for explanations. 

//Include the U8g2 library
#include <U8g2lib.h>
#include <U8x8lib.h>

/*Those are the pins the Arduino connects to the display with.
 * Find more details regarding the pins: http://www.newhavendisplay.com/specs/NHD-C12864A1Z-FSB-FBW-HTT.pdf
 */
#define LCD_SERIAL_CLOCK_PIN 10
#define LCD_SERIAL_DATA_PIN 9
#define LCD_REGISTER_SELECT_PIN 8
#define LCD_RESET_PIN 7
#define LCD_CHIP_SELECT_PIN 6


/*The U8g2 library requires a display object. in this case, it's called "u8g2".
 * For more information on how to use the library: https://github.com/olikraus/u8g2/wiki/setup_tutorial
 */
U8G2_ST7565_NHD_C12864_F_4W_SW_SPI u8g2(U8G2_R0, LCD_SERIAL_CLOCK_PIN, LCD_SERIAL_DATA_PIN, LCD_CHIP_SELECT_PIN, LCD_REGISTER_SELECT_PIN, LCD_RESET_PIN);

void setup() {
  //Start the lcd communication
  u8g2.begin();
}

void loop() {
  //Shows the text
  u8g2.firstPage();

  do {
    /*Sets the font.
     * List of all the fonts: https://github.com/olikraus/u8g2/wiki/fntlistall#all-u8g2-fonts-capital-a-height
     */
    u8g2.setFont(u8g2_font_ncenB14_tr);

    //Draws a string at position X = 0 and Y = 24. Play around with the text!
    u8g2.drawStr(0, 24, "Hello World!");
  } while ( u8g2.nextPage() );
}
